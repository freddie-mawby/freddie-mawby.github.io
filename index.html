About Me

Hello! Maps


This page features the maps I've made for my GIS course.
SDG Indicator 15.3.1 Choropleth Map (static)

This is a map of sustainable development goal indicator 15.3.1, which shows the percentage of degraded land for each country.
Made in QGIS, using data from the UNCCD.

SDG Indicator 15.3.1 Choropleth Map (again, but interactive!)

Using the shapefile I got from the previous map, I made a web app with ArcGIS Online services. This one is interactive, you can click on any country and see its percentage of degraded land. The legend is also interactive, you can select any category and see which countries fall into it.
(You do need an Esri account to see it though...)

Snow Coverage Arve Valley (October 2021-March 2022)

This map is very personal to me, since I grew up in the Arve Valley (French Alps)!
This one was made using Google Earth Engine, and it highlights the snow coverage in that area using false color composite. The different layers show monthly changes between October 2021 and March 2022.
All data is from Sentinel 2 satellite.
